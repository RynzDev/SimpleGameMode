### SimpleGameMode

Simple Gamemode Plugin :D

### Example

/gmc RynzDev

### Commands
|**Command**|**Description**|
|-----------|---------------|
|`/gms`|Survival Mode|
|`/gmc`|Creative Mode|
|`/gma`|Adventure Mode|
|`/gmspc`|Spectator Mode|
